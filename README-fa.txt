راهنمای اجرای نسخه داخلی سایت علامه سخن

این نسخه دیگر به Firebase، CDN فونت، Tawk یا پلتفرم خارجی وصل نیست.
فایل index.html ظاهر و پنل سایت است و فایل server.js سرور داخلی ساده برای ذخیره کاربران، پیام ها، نمره ها و تاریخ آزمون هاست.

روش اجرا روی سرور خودتان:

1. Node.js را روی سرور نصب کنید.
2. کل پوشه outputs را روی دیتاسنتر/سرور خودتان قرار دهید.
3. داخل همین پوشه این دستور را اجرا کنید:

   node server.js

4. سایت از این آدرس بالا می آید:

   http://YOUR-SERVER-IP:8080

5. حساب ادمین اولیه:

   ایمیل: admin@allameh-sokhan.ir
   رمز عبور: 123456

بعد از اولین ورود، رمز عبور ادمین را از پنل تغییر دهید.
داده ها در فایل data/store.json کنار سایت ذخیره می شوند و روی سرور خودتان باقی می مانند.

برای تغییر پورت:

   PORT=3000 node server.js

در ویندوز:

   $env:PORT=3000; node server.js
