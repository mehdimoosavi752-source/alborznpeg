# راه‌اندازی آزمایشی روی Render

این پوشه نسخه مخصوص Render است و از Node.js و PostgreSQL استفاده می‌کند.

1. تمام فایل‌های داخل این پوشه را در ریشه مخزن GitHub بارگذاری کنید.
2. در Render گزینه `New` و سپس `Blueprint` را انتخاب کنید.
3. مخزن `Allameh-Sokhan-site` را انتخاب کنید.
4. هنگام ساخت Blueprint برای `ADMIN_PASSWORD` یک رمز حداقل ۸ کاراکتری وارد کنید.
5. ساخت سرویس را تأیید کنید و تا نمایش وضعیت `Live` منتظر بمانید.
6. آدرس سایت از صفحه سرویس Web در دسترس خواهد بود.

ایمیل مدیر به‌صورت پیش‌فرض:

`admin@asmdi.ir`

هشدار: دیتابیس رایگان Render برای آزمایش است و طبق شرایط فعلی Render پس از ۳۰ روز منقضی می‌شود.
