# نصب کامل سایت و پنل علامه سخن روی هاست

این نسخه به Node.js، Docker، Firebase یا سرویس خارجی نیاز ندارد و با PHP و MySQL هاست شما اجرا می‌شود.

## نیازمندی‌ها

- PHP نسخه 8.1 یا بالاتر
- MySQL 5.7 یا MariaDB 10.3 به بالا
- افزونه PHP PDO MySQL
- HTTPS فعال
- Apache یا LiteSpeed با پشتیبانی از `.htaccess`

## نصب

1. در DirectAdmin یا cPanel یک دیتابیس MySQL، نام کاربری و رمز دیتابیس بسازید.
2. فایل `config.php` را باز کنید و این چهار مقدار را وارد کنید:

```php
'host' => 'localhost',
'name' => 'نام دیتابیس',
'user' => 'نام کاربری دیتابیس',
'pass' => 'رمز دیتابیس',
```

3. در همان فایل، ایمیل و رمز اولیه مدیر را تغییر دهید:

```php
'email' => 'ایمیل مدیر',
'password' => 'یک رمز قوی و غیرقابل حدس',
```

4. محتویات بسته را داخل `public_html` آپلود و Extract کنید.
5. آدرس زیر را باز کنید:

```text
https://YOUR-DOMAIN/api/health
```

اگر نصب صحیح باشد، این پاسخ دیده می‌شود:

```json
{"ok":true}
```

6. سایت را باز کنید و با ایمیل و رمز مدیر وارد شوید. جدول‌های دیتابیس و حساب مدیر در اولین اجرا خودکار ساخته می‌شوند.

## قابلیت‌های فعال

- ثبت‌نام و ورود دانشجو
- پروفایل، شماره موبایل و تنظیمات تأیید موبایل
- پنل ادمین و فهرست دانشجویان
- ثبت‌نام دوره، آزمون اصلی، آزمون Mock و تعیین سطح
- مشاهده مجزای ثبت‌نام‌ها توسط ادمین
- پیشنهاد دوره یا آزمون از طرف ادمین به دانشجو
- مدیریت تاریخ‌های TOEFL، Mock TOEFL و GRE
- ساخت آزمون داخلی و ثبت پاسخ نوشتاری و صوتی
- پیام‌های دانشجو و مدیریت
- ثبت نمرات چهار مهارت

## تنظیم آپلود و ضبط صدا

در بخش PHP Settings هاست این مقادیر پیشنهاد می‌شوند:

```ini
post_max_size = 32M
upload_max_filesize = 32M
memory_limit = 256M
max_execution_time = 120
```

مرورگر فقط روی دامنه دارای HTTPS اجازه استفاده از میکروفن را می‌دهد.

## نکات امنیتی

- فایل `config.php` توسط `.htaccess` از دسترسی مستقیم محافظت شده است.
- بعد از اولین ورود، رمز مدیر را از داخل پنل تغییر دهید.
- از دیتابیس به‌صورت منظم Backup بگیرید.
- فایل `.htaccess` و پوشه `api` را حذف یا تغییرنام ندهید.

## نصب در زیردامنه

بهتر است بسته را مستقیماً در ریشه دامنه یا زیردامنه قرار دهید؛ برای نمونه:

```text
public_html/
```

یا:

```text
domains/panel.example.com/public_html/
```
