پیشنهادهای لوگو

چهار مسیر پیشنهادی برای هویت بصری نوین پی تست البرز؛ گزینه اول روی سایت به‌عنوان نسخه پیشنهادی فعال شده است.

فعال روی سایت

پی و سازه

مناسب‌ترین گزینه برای تاکید روی عمران، استحکام و هویت حرفه‌ای.

جرثقیل و اجرا

نمایشی‌تر، مناسب برندهایی که می‌خواهند حس پروژه و کارگاه پررنگ باشد.

نقشه و مهندسی

مینیمال‌تر و مناسب تاکید روی طراحی، نقشه، کنترل و مستندسازی.

مونوگرام لوکس

ساده، برندپذیر و خوب برای مهر، سربرگ، کارت و شبکه‌های اجتماعی.
